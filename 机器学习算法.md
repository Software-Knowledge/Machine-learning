# 1. 算法分类
1. 第一种分类方式:
    1. 分类算法:基本都是有监督的，类别可以看做导致信号
    2. 聚类算法
    3. 回归算法:数值型变量之间的关系
    4. 关联算法:非监督算法
2. 第二种分类方式:有没有导致信号
    1. 监督算法:有信号
    2. 非监督算法:无信号

# 2. 数据可视化
1. 重要的算法
2. 在机器学习中的作用:
    1. 视觉是人类获得信息的最主要途径
    2. 可视化本身是一种机器学习算法
    3. 可视化可以作为数据预处理的方法或者是机器学习过程的表示方法
    4. 机器学习的结果也可以用可视化形式来表示
3. 案例:可以排除噪声点、大致判断关系，促进选择算法。
