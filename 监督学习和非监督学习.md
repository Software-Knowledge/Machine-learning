监督学习和非监督学习
---

# 1. 监督学习
1. 给定包含正确答案的数据集，用来给出更多的正确答案。
2. 通常被我们成为回归问题(连续性数据)
3. 分类问题:离散型输出
   1. 使用不同的符号绘制图形
   2. 使用直线划分二分类问题
   3. 能够处理多特征的算法是很有用的

# 非监督学习
1. 无标签，无目标，只有数据集，进行自动分簇
2. 没有提前知道算法
3. 无监督学习的例子(聚类)
   1. 被用于大规模计算机集群
   2. 对于客户，进行细分市场进行销售
   3. 天文数据分析
4. 其他的用法:
   1. 麦克风问题:距离不同
   2. 分离这两个音频，成完整的音频
   3. 实现如下

```
[W,s,v] = svd((repmat(sum(x.*x,1),size(x,1).*x)*x'))
```
